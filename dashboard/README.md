# Curbside Social Co. Dashboard

## File structure
- index.html              main app shell + all tab HTML
- css/main.css            layout, topbar, tabs, grid, responsive breakpoints
- css/components.css      cards, badges, forms, pipeline, calendar, clients
- js/tabs.js              hash-based routing (#outreach, #leads, etc.)
- js/outreach.js          outreach message generator
- js/projects.js          checklist toggle
- assets/                 logo, favicon, images

## Deploy to GitHub Pages
1. Create repo at github.com/curbsidesocialco/dashboard
2. Push all files
3. Settings > Pages > source: main branch, / (root)
4. Live at: https://curbsidesocialco.github.io/dashboard
